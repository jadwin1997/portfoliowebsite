LADDER Geospatial Data Analysis App

LADDER is a data analysis app created with the goal in mind to assist farmers in making data-driven decisions by using their own farms' harvest data.
LADDER includes an autonomous John Deere downloader that utilizes John Deere's API to continously gather geospatial harvest data as farm operations were completed. Users could then use this data to aggregate/rasterize data points, perform spatial joins around soil sample points, and create topology maps of their fields.
LADDER has a data model generation feature where users can define features and target columns for their data and generate a data model that users can use to predict future yields and improve crop viability.
LADDER included other features such as auto report generation, user-friendly data plotting, and soil data enrichment (using web soil surveys API).

Mobile PFAS Detection Rover

An Ardurover based vehicle capable of transition between water and land seamlessly. This rover was created to autonomously test bodies of water for microplastics.
A water intake system and water testing system was designed that kept the rovers sterile water control samples separate from test samples to avoid cross contamination.
A potentiostat was then used in conjunction with a functionlized polymer electrode to determine if microplastics were present.

Automated Logos Autonomous Field Painting Rover

An Ardurover based vehicle capable of painting fields for recreational sports events. This was for my final capstone senior design project in the Electrical and Computer Engineering department.
Automated Logos utilized an Ardurover flight controller and a RaspberryPi single board computer to give users an intuitive way to autonomously paint a large field. The user created their design using primative shapes. These shapes were then uploaded to the flight controller and converted into waypoints for an autonomous mission.
Automated Logos used an RTK enabled GPS and was capable of centimeter level accuracy.

IEEE SECON Hardware Competition 2023

An autonomous mobile robot capable of sweeping up rubber ducks and depositting them into a predetermined area.

This robot utilized a sensor fusion algorithm that fused the MPU, front IR sensor, and side IR sensors together using a complementary filter to estimate it's heading as well as its position on the field.
A "stuck-detection" function was created for this robot so it would accelerate forward to offset speed losses due to obstacles on the course.
This had the added benefit of moving the robot out of local minimums and maximums.
